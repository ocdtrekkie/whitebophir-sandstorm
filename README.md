# WBO for Sandstorm

[WBO](https://github.com/lovasoa/whitebophir) is an online collaborative whiteboard that allows many users to draw simultaneously on a large virtual board.
The board is updated in real time for all connected users, and its state is always persisted. It can be used for many different purposes, including art, entertainment, design, teaching.

Note: SVG download on Sandstorm requires opening the app frame in a new tab.
